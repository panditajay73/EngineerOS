# Installation Guide

EngineerOS behaves like a modern mobile app after installation.

## Android Chrome

1. Open EngineerOS.
2. Tap the floating `Install StudyOS` button or the browser menu.
3. Choose Install app.
4. Launch EngineerOS from the home screen.

## Samsung Internet

1. Open EngineerOS.
2. Open the browser menu.
3. Choose Add page to or Install web app.
4. Add it to the home screen.

## Desktop Chrome

1. Open EngineerOS.
2. Click the floating `Install StudyOS` button or the install icon in the address bar.
3. Confirm installation.

## Desktop Edge

1. Open EngineerOS.
2. Click the floating `Install StudyOS` button or Apps -> Install this site as an app.
3. Confirm installation.

## iPhone Safari

Safari on iPhone does not provide the automatic install prompt used by Chrome and Edge.

Use:

1. Open EngineerOS in Safari.
2. Tap Share.
3. Tap Add to Home Screen.
4. Confirm the name and tap Add.

## Installed App Behavior

After installation:

- The app opens standalone.
- Browser chrome is hidden.
- Theme color and icons are applied.
- Offline support works after the first successful load.
